The classic logic puzzle game.

How to Play

  1. The objective is to fill a 9x9 grid with digits so that each column, each row, and each of the nine 3x3 subgrids that compose the grid contain all of the digits from 1 to 9.
  2. The puzzle starts with a partially completed grid, which has a single unique solution.
  3. Fill in the empty cells with numbers from 1 to 9.
  4. No number can be repeated in the same row, column, or 3x3 square.
